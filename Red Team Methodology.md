## Red Team Methodology

1. Reconnaissance
   - Network scanning
   - Web surface mapping
   - Service fingerprinting

2. Initial Access
   - Web exploitation
   - Credential abuse
   - Misconfiguration exploitation

3. Post-Exploitation
   - Local enumeration
   - Credential harvesting
   - Access stabilization

4. Lateral Movement
   - SSH / SMB / WinRM
   - Port forwarding
   - AD abuse

5. Privilege Escalation
   - Linux sudo/SUID abuse
   - Windows token impersonation
   - Service misconfiguration

6. Impact & Reporting
   - Proof of compromise
   - Risk articulation
   - Defensive recommendations
